# FinTrack Mobile

Мобільний застосунок для відстеження особистих фінансів — доходів, витрат та загальної статистики транзакцій.

## Про застосунок

FinTrack Mobile дозволяє користувачам вести облік своїх фінансів у зручному форматі: додавати транзакції, категоризувати їх та аналізувати свої витрати й доходи через наочну статистику.

## Функціональні можливості

- **Транзакції** — додавання, редагування та видалення записів про доходи та витрати
- **Категорії** — розподіл транзакцій за категоріями (їжа, транспорт, розваги, зарплата тощо)
- **Статистика** — графіки та зведення витрат і доходів за обраний період
- **Фільтрація** — перегляд транзакцій за датою, типом або категорією
- **Баланс** — відображення поточного балансу в реальному часі

## Технічний стек

- **React Native** — кросплатформна розробка (iOS / Android)
- **Expo** — середовище розробки та збірки
- **React Navigation** — навігація між екранами
- **AsyncStorage / SQLite** — локальне зберігання даних

## Встановлення та запуск

### Вимоги

- Node.js >= 18
- npm або yarn
- Expo CLI

### Кроки

```bash
# Клонувати репозиторій
git clone <repo-url>
cd FinTrack-Mobile

# Встановити залежності
npm install

# Запустити застосунок
npx expo start
```

Після запуску відскануйте QR-код у додатку **Expo Go** (iOS / Android) або запустіть емулятор.
